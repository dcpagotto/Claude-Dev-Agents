## How to Use

Each example shows the full agent orchestration flow:
1. User request
2. Tech-lead analysis and agent routing
3. Step-by-step agent execution with context passing
4. Final result

These examples demonstrate the three-phase workflow (Research, Planning, Execution) described in the main documentation.
